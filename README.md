## Project Overview

This project consists of a static professional portfolio website built with **HTML, CSS, and JavaScript**, version-controlled with **Git and GitHub**, and deployed automatically using a **CI/CD pipeline with GitHub Actions**.

The website is hosted on **Azure Static Web Apps** with a **custom domain**, and downloadable PDF files (CV in ES/EN) are securely served from **Azure Blob Storage**.

The goal of this project is to showcase both **technical skills** and **cloud-based deployment practices** in a real-world scenario.

---

## Project Objectives

- Design and develop a clean, professional static website
- Implement version control and collaborative workflows with Git & GitHub
- Automate build and deployment processes using CI/CD
- Deploy and host the site using Azure cloud services
- Integrate secure file distribution for downloadable documents
- Apply best practices for structure, basic SEO, and multilingual support

## Architecture Overview

- **Source Code**: Managed and versioned using GitHub  
- **CI/CD Pipeline**:  
  - Triggered on push to the main branch  
  - Automated build and deployment via GitHub Actions  
- **Hosting**:  
  - Website hosted on Azure Static Web Apps  
- **File Distribution**:  
  - PDF documents (CV ES/EN) stored in Azure Blob Storage  
  - Download links integrated into the website
