Portafolio Web Personal en Azure

Descripción del proyecto

Diseño, implementación y despliegue de éste sitio web utilizando tecnologías web y servicios cloud de Azure, integrando control de versiones y automatización de despliegues mediante CI/CD.

Objetivo

Diseñar, implementar y desplegar un sitio web estático profesional utilizando HTML, CSS y JavaScript, versionado con Git y GitHub, e integrado a un flujo de CI/CD con GitHub Actions para automatizar el proceso de build y despliegue.
El sitio se encuentra hospedado en Azure Static Web Apps, con dominio personalizado, y utiliza Azure Storage (Blob Storage) para la distribución segura de archivos PDF descargables en español e inglés.


Alcance del proyecto

  • Desarrollo de sitio web estático con HTML, CSS y JavaScript.
  • Control de versiones y gestión de cambios con Git y GitHub.
  • Implementación de pipeline CI/CD con GitHub Actions para despliegue automático.
  • Hosting del sitio en Azure Static Web Apps.
  • Configuración de dominio personalizado y enlace con el servicio web.
  • Creación y administración de Azure Storage Account (Blob Storage) para distribución de archivos PDF.
  • Integración de enlaces de descarga multi-idioma (ES / EN) desde Azure Blob Storage.
  • Aplicación de buenas prácticas de estructura, SEO básico y separación de entornos.

Diseño e Implementación de Infraestructura de Red

Descripción del proyecto

Diseño e implementación de una infraestructura de red empresarial basada en un modelo jerárquico de tres capas (Acceso, Distribución/Core y Perímetro), integrando switching, routing dinámico y seguridad perimetral.
El laboratorio simula un entorno corporativo real con alta disponibilidad, segmentación de red y salida segura a Internet.

Objetivo

Construir una topología escalable y robusta que permita: Comunicación eficiente entre segmentos de red, Enrutamiento dinámico interno y externo y Control y protección del tráfico hacia Internet.

Componentes y tecnologías utilizadas

Capa de Acceso (L2)
  • Cisco Catalyst 2960-X y 2960-S
  • Trunking VLAN y STP
Capa de Distribución / Core (L3)
  • Cisco Catalyst 3750-X
  • SVIs y enrutamiento OSPF (Área 0)
Routers
  • Cisco 1900 y Cisco 1800
  • OSPF para routing interno
  • BGP para interconexión entre sistemas autónomos
Seguridad Perimetral
  • Firewall Cisco ASA 5508-X
  • BGP con el core
  • Segmentación Inside / Outside
  • Salida a Internet mediante DHCP

Alcance del proyecto

  • Implementación de OSPF como protocolo de enrutamiento interno.
  • Uso de BGP para interconexión entre routers y firewall, simulando un entorno multi-AS.
  • Firewall ASA actuando como punto de salida a Internet.
  • Diseño modular que facilita escalabilidad y troubleshooting.
  • Laboratorio documentado con configuraciones reales de dispositivos Cisco
